转--忘记xampp的phpmyadmin登录密码后如何重置密码

Share

#前言
xampp集成开发环境，包含有phpmyadmin数据库管理工具，可以登录以管理mysql数据库，但有时因误操作，比如删除了localhost的账户名，或因忘记了登录帐密，会被拒绝登入。
在谷歌、百度多种方法后，终于发现以下这种方法，亲测可用

#正文

  1. 首先进入DOS，开一个cmd,关闭mysql（有的会提示未安装服务，只要关闭mysql的运行即可，不影响后续操作）

    c:>sc stop mysql

  2. 进到mysql的bin目录下：(根据mysql所在盘,修改为对应的路径)

    cd C:\xampp\mysql\bin

  3. 执行命令(输入以下命令后，命令行不可动，不关闭，另开一个cmd界面)

    mysqld –defaults-file=”C:\xampp\mysql\bin\my.ini” –console –skip-grant-tables

  4. 另开一个cmd，进到mysql的bin目录下

    cd C:\xampp\mysql\bin

  5. 输入命令

    C:\xampp\mysql\bin>mysql -uroot mysql

    mysql>UPDATE user SET Password=PASSWORD(‘123456’) where USER=’root’

    mysql> FLUSH PRIVILEGES

    mysql> quit

    C:\xampp\mysql\bin>mysqladmin shutdown

  6. 最后，重启mysql服务器。

###需要注意的是，要核对代码、空格和引号
