用Axure设计动态展示图

Share

#前言
近期开始接触产品设计方面的知识，在朋友的推荐下，使用了这一款软件–>Axure

关于Axure的介绍和入门，网上有着丰富的资源，这里不再赘述，分享一个刚学的“图片自动切换”（幻灯片播放）。我的学习资源来自–>Axure原创教程网


##方法步骤
1.在编辑区域拉入“动态面板”，命名为“Panel”。双击面板，添加5个面板状态State1-5



2.给每个面板状态导入一张图片




3.为动态面板Panel添加用例“载入时”，设置面板状态，配置动作中勾选Panel,选择状态“Next”,勾选“向后循环”、“循环间隔”，选择进入动画“向左滑动”




4.接着，制作画面下方随图片变化时的小圆点。

（1）引入矩形,设置为圆角矩形，作为圆点的背景；

（2）制作5个圆形，分别命名后，按住CRTL键选择5个原形后，设置选项组名称为Roll；



5.接下来设置图片切换时，小圆点随着变化
（1）为动态面板Panel，添加“状态改变时”用例，添加条件：“if面板状态于This == State1”;



（2）继续在“状态改变时”面板选择“选中”，配置动作点选“tag01”，确定。



6.接着继续为动态面板Panel添加“状态改变时”用例，设置State2-5.



完成步骤6后，进入预览，图片可实现自动播放

7.设置图片切换时，小圆点的变化

选择tag01，在交互样式设置中选择”选中”，设置填充颜色；同样给tag02-05设置一样的步骤。预览，画面切换，小圆点跟随变化。


将tag01设置为预览时就显示

转--忘记xampp的phpmyadmin登录密码后如何重置密码

Share

#前言
xampp集成开发环境，包含有phpmyadmin数据库管理工具，可以登录以管理mysql数据库，但有时因误操作，比如删除了localhost的账户名，或因忘记了登录帐密，会被拒绝登入。
在谷歌、百度多种方法后，终于发现以下这种方法，亲测可用

#正文

  1. 首先进入DOS，开一个cmd,关闭mysql（有的会提示未安装服务，只要关闭mysql的运行即可，不影响后续操作）

    c:>sc stop mysql

  2. 进到mysql的bin目录下：(根据mysql所在盘,修改为对应的路径)

    cd C:\xampp\mysql\bin

  3. 执行命令(输入以下命令后，命令行不可动，不关闭，另开一个cmd界面)

    mysqld –defaults-file=”C:\xampp\mysql\bin\my.ini” –console –skip-grant-tables

  4. 另开一个cmd，进到mysql的bin目录下

    cd C:\xampp\mysql\bin

  5. 输入命令

    C:\xampp\mysql\bin>mysql -uroot mysql

    mysql>UPDATE user SET Password=PASSWORD(‘123456’) where USER=’root’

    mysql> FLUSH PRIVILEGES

    mysql> quit

    C:\xampp\mysql\bin>mysqladmin shutdown

  6. 最后，重启mysql服务器。

###需要注意的是，要核对代码、空格和引号
